# Blackjack Game

Welcome to the Blackjack Game! The goal is to get as close to 21 as possible without going over.

## Files

- `index.html`: The main HTML file containing the structure of the game.
- `style.css`: The CSS file for styling the game interface.
- `main.js`: The JavaScript file containing the game logic.

## How to Play

1. Open `index.html` in a web browser.
2. Click **New Game** to start.
3. Use the **Hit** button to draw a card or the **Stand** button to stop drawing.
4. Try to beat the dealer without exceeding 21 points!

## Requirements

- A modern web browser (e.g., Chrome, Firefox, Edge).
- No additional software or dependencies are required.

## How to Run the Project

1. Clone or download the project files.
2. Open `index.html` in your browser.
3. Play and enjoy!
